Somni Systems Website

This repository contains the official website for Somni Systems.

Somni Systems is an early-stage startup developing personalized sleep support solutions to help people find the right level of neck support and improve sleep quality.

The website provides information about the company, our mission, the problem we are addressing, and our product vision. It also allows visitors to join our waitlist and follow our development journey.

Website: https://somni-systems.vercel.app
